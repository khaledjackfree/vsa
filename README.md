# 🎮 لعبة XO - Tic Tac Toe v2.0

لعبة إكس-أو (Tic Tac Toe) كلاسيكية مع واجهة عصرية، ذكاء اصطناعي متعدد المستويات، ومؤثرات صوتية وبصرية رائعة.

## ✨ المميزات

### 🎯 أوضاع اللعب
- 👥 **وضع لاعبين (PvP)** - العب مع صديقك على نفس الجهاز
- 🤖 **ضد الكمبيوتر (سهل)** - حركات عشوائية للمبتدئين
- ⚡ **ضد الكمبيوتر (متوسط)** - مزيج من الذكاء والعشوائية
- 🔥 **ضد الكمبيوتر (صعب)** - ذكاء اصطناعي بخوارزمية **Minimax** (لا يمكن هزيمته!)

### 🎨 الواجهة والتصميم
- 🌓 **وضع ليلي/نهاري** - بدّل بين الثيمين بضغطة زر (محفوظ تلقائياً)
- 🎨 **تصميم عصري** بألوان متدرجة وتأثيرات Glass Morphism
- ✨ **رسوم متحركة سلسة** لكل حركة وخانة فائزة
- 🎉 **تأثير Confetti** عند الفوز
- 🏆 **مودال احتفالي** يظهر بعد انتهاء كل جولة
- 📱 **متجاوبة** تعمل على الموبايل، التابلت، والكمبيوتر
- 🌍 **واجهة عربية** بدعم RTL

### 🎵 المؤثرات الصوتية
- 🔊 أصوات نقر لكل حركة
- 🎵 نغمة فوز احتفالية (3 نوتات صاعدة)
- 🔔 نغمة تعادل
- 🔇 إمكانية إيقاف الصوت بضغطة زر (محفوظ تلقائياً)

### 📊 الإحصائيات والسجلات
- 📊 **لوحة نتائج** تحفظ عدد مكاسب كل لاعب والتعادلات
- 📜 **سجل الحركات** يعرض كل حركة بترتيبها وموقعها
- ↩️ **زر التراجع (Undo)** للرجوع عن آخر حركة
- 💾 **حفظ تلقائي** للنتائج، الثيم، الأسماء، وإعدادات الصوت في localStorage

### 👤 تخصيص شخصي
- 📝 **إدخال أسماء اللاعبين** المخصصة
- 🔄 **تبديل من يبدأ** كل جولة تلقائياً بالعدل

### ⌨️ اختصارات لوحة المفاتيح
| المفتاح | الوظيفة |
|---------|---------|
| `1` - `9` | وضع علامة في الخانة المقابلة |
| `R` | جولة جديدة |
| `U` | تراجع عن آخر حركة |
| `T` | تبديل الثيم (ليلي/نهاري) |
| `S` | تشغيل/إيقاف الصوت |
| `Enter` | جولة جديدة (من المودال) |
| `Esc` | إغلاق المودال |

## 🎯 كيفية اللعب

1. **(اختياري)** أدخل أسماء اللاعبين في الحقول العلوية
2. اختر وضع اللعب: **لاعبين**، **سهل**، **متوسط**، أو **صعب**
3. اللاعب **X** يبدأ الجولة الأولى (ثم يتبادل الدور في كل جولة جديدة)
4. انقر على أي خلية فارغة أو استخدم الأرقام **1-9** على لوحة المفاتيح
5. أول من يصفّ 3 متتالية (أفقي / عمودي / قطري) يفوز 🎉
6. استخدم **تراجع ↩️** لإلغاء آخر حركة
7. اضغط **جولة جديدة** للعب من جديد

## 🧠 خوارزمية الذكاء الاصطناعي

### الوضع السهل 🤖
- يختار حركة عشوائية من الخانات الفارغة.

### الوضع المتوسط ⚡
- 50% من الوقت يلعب بذكاء (Minimax)
- 50% من الوقت يلعب عشوائياً
- مناسب للتحدي المتوازن

### الوضع الصعب 🔥
يستخدم خوارزمية **Minimax** الكلاسيكية التي:
- تحسب كل الحركات الممكنة وتخيلات المستقبل recursively
- تختار دائماً الحركة المثالية لتعظيم فرصها وتقليل فرص الخصم
- **مستحيل هزيمتها** — أفضل ما يمكنك تحقيقه هو **التعادل**! 🤖

## 🛠️ التقنيات المستخدمة

- **HTML5** - الهيكل الدلالي
- **CSS3** - التصميم (Grid, Flexbox, Animations, CSS Variables, Backdrop Filter)
- **JavaScript (Vanilla ES6+)** - المنطق والذكاء الاصطناعي
- **Web Audio API** - توليد المؤثرات الصوتية برمجياً
- **LocalStorage API** - حفظ النتائج والإعدادات
- **CSS Custom Properties** - دعم الثيمات (ليلي/نهاري)
