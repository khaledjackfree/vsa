# 🎮 لعبة XO - Tic Tac Toe

لعبة إكس-أو (Tic Tac Toe) كلاسيكية مع واجهة عصرية وذكاء اصطناعي.

## ✨ المميزات

- 🎨 **تصميم عصري** بألوان متدرجة وتأثيرات بصرية جميلة
- 👥 **وضع لاعبين** (PvP) - العب مع صديقك على نفس الجهاز
- 🤖 **ضد الكمبيوتر (سهل)** - حركات عشوائية للمبتدئين
- 🔥 **ضد الكمبيوتر (صعب)** - ذكاء اصطناعي بخوارزمية **Minimax** (لا يمكن هزيمته!)
- 📊 **لوحة نتائج** تحفظ عدد المكاسب والتعادلات
- 💾 **حفظ تلقائي** للنتائج في المتصفح (localStorage)
- 📱 **متجاوبة** تعمل على الموبايل والكمبيوتر
- 🌍 **واجهة عربية** بدعم RTL

## 🚀 كيفية التشغيل

### محلياً:
1. استنسخ الريبو:
   ```bash
   git clone https://github.com/khaledjackfree/vsa.git
   cd vsa
   ```
2. افتح `index.html` في أي متصفح.

### أو بسيرفر محلي:
```bash
python -m http.server 8000
```
ثم افتح: `http://localhost:8000`

## 🎯 كيفية اللعب

1. اختر وضع اللعب: **لاعبين** أو **سهل** أو **صعب**
2. اللاعب **X** يبدأ دائماً
3. انقر على أي خلية فارغة لوضع علامتك
4. أول من يصفّ 3 متتالية (أفقي / عمودي / قطري) يفوز 🎉
5. اضغط **جولة جديدة** للعب من جديد
6. اضغط **مسح النتائج** لتصفير لوحة النتائج

## 🧠 خوارزمية الذكاء الاصطناعي

الوضع الصعب يستخدم خوارزمية **Minimax** التي:
- تحسب كل الحركات الممكنة وتخيّلات المستقبل
- تختار دائماً الحركة المثالية
- مستحيل هزيمتها — أفضل ما يمكنك تحقيقه هو **التعادل**! 🤖

## 🛠️ التقنيات المستخدمة

- **HTML5** - الهيكل
- **CSS3** - التصميم (Grid, Flexbox, Animations, Backdrop Filter)
- **JavaScript (Vanilla)** - المنطق والذكاء الاصطناعي
- **LocalStorage API** - حفظ النتائج
